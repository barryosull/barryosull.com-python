Each iframe uses in-memory storage to simulate 5 separate players in the same browser

How to Test:

  1. Make sure the backend is running on http://localhost:8000
  2. Make sure the frontend dev server is running: npm run dev
  3. Click "Reload All Players" to start fresh
  4. In Player 1, create a new game room and note the Room ID
  5. In Players 2-5, join the room using that Room ID
  6. In Player 1, start the game once all 5 players have joined
  7. Play through the game mechanics in each player's window
